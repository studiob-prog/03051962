GUÍA DE CONFIGURACIÓN

Demon Hunters K-Pop · Sistema de Boletos
Tiempo estimado: 20-30 minutos. Sigues esta guía una sola vez y el sistema queda listo para todas las funciones.
PASO 01
Crear proyecto Firebase GRATIS

Firebase es la base de datos que guarda los boletos vendidos en tiempo real. El plan gratuito es más que suficiente para tu evento.

  • Ve a console.firebase.google.com
  • Clic en "Agregar proyecto" → ponle nombre (ej: demon-hunters-kpop)
  • Desactiva Google Analytics (no es necesario) → "Crear proyecto"
  • Espera ~30 segundos hasta que diga "Tu nuevo proyecto está listo"
PASO 02
Activar Firestore Database

Firestore es la base de datos donde se guardan boletos e inventario.

  • En el menú izquierdo de Firebase → "Firestore Database"
  • Clic en "Crear base de datos"
  • Selecciona "Comenzar en modo de prueba" (suficiente para el evento)
  • Elige la región us-central1 o la más cercana a México → "Habilitar"

Reglas de seguridad — En Firestore → pestaña "Reglas", pega esto y publica:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { // Cualquiera puede leer inventario match /kpop_show/inventory { allow read: true; allow write: true; } // Cualquiera puede crear y leer órdenes propias match /kpop_show/orders/{orderId} { allow read, write: true; } } }
Nota: Estas reglas son permisivas para facilitar el setup. Para mayor seguridad, puedes restringirlas después del evento.
PASO 03
Obtener las credenciales de Firebase
  • En el panel de Firebase → ícono de engranaje ⚙ → "Configuración del proyecto"
  • Baja hasta "Tus apps" → clic en el ícono </> (Web)
  • Ponle un apodo (ej: boletos-web) → "Registrar app"
  • Copia el bloque firebaseConfig que aparece. Se ve así:
const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "demon-hunters.firebaseapp.com", projectId: "demon-hunters-kpop", storageBucket: "demon-hunters-kpop.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abc123" };

Pega este bloque reemplazando el que viene en ambos archivos: demon-hunters-boletos.html y demon-hunters-admin.html

✓ Una vez pegado, Firebase queda conectado. Los boletos se guardan automáticamente.
PASO 04
Crear Links de Pago en MercadoPago MP

Necesitas crear 3 links de pago (uno por zona). MercadoPago cobrará una comisión de ~3.49% por transacción aprobada.

  • Entra a mercadopago.com.mx con tu cuenta de negocio
  • Ve a Cobrar → Cobros en línea → Link de pago
  • Selecciona "Monto variable" (para que el total cambie según qty y descuento)
  • Crea 3 links y nómbralos: Boleto VIP, Boleto Preferente, Boleto 2do Piso
  • Copia cada URL y pégala en el archivo demon-hunters-boletos.html en la sección MP_LINKS
const MP_LINKS = { vip: "https://mpago.la/TU_LINK_VIP", pref: "https://mpago.la/TU_LINK_PREF", seg: "https://mpago.la/TU_LINK_SEG", };
URL de retorno: En la configuración de cada link de MercadoPago, configura la URL de retorno apuntando a donde subas tu archivo. Por ejemplo:
https://tudominio.com/boletos.html
Esto hace que después del pago el comprador vuelva a ver su QR automáticamente.
PASO 05
Configurar tus Códigos de Descuento

Edita este bloque en demon-hunters-boletos.html para cambiar el código o el porcentaje cuando lo necesites:

const DISCOUNT_CODES = [ { code: "KPOP2025", pct: 20, label: "20% de descuento" }, // Para agregar más en el futuro, descomenta y edita: // { code: "OTRO_CODIGO", pct: 15, label: "Descripción" }, ];

El código es insensible a mayúsculas — el usuario puede escribir kpop2025 o KPOP2025 y funciona igual.

PASO 06
Publicar los archivos (hosting gratuito)

La forma más fácil es usar Netlify Drop — sin registro necesario:

  • Ve a app.netlify.com/drop
  • Arrastra los 3 archivos HTML a la zona de drop
  • En segundos te da una URL pública tipo magical-name-123.netlify.app
  • Esa URL la compartes en redes, WhatsApp y con las escuelas
  • El admin queda en tu-url.netlify.app/demon-hunters-admin.html
Tip: Para tener una URL personalizada (ej: demonhunters.netlify.app), crea una cuenta gratis en Netlify y puedes elegir el nombre del subdominio.
PASO 07
Usar el Panel de Administrador

El archivo demon-hunters-admin.html te da acceso a:

  • KPIs en tiempo real: Ingreso total, boletos vendidos, % de ocupación
  • Disponibilidad por función y zona con barras de progreso
  • Tabla de todas las órdenes con filtros por función, zona y estado
  • Exportar CSV para llevar tu contaduría en Excel

La contraseña por defecto es demonhunters2025. Cámbiala en el archivo buscando:

const ADMIN_PASSWORD = "demonhunters2025"; // ← cambia esto
✓ El panel se actualiza en tiempo real — cada vez que alguien compra, lo ves aparecer al instante sin recargar la página.
